游戏介绍

在“北京浮生记”中，您扮演一位从外地到北京谋生的青年。一开始，您只有2000元钱，并且还欠村长（一个流氓头子）5400元 债务。这些债务每天的利息高达10%.如果不及时还清，村长会叫在北京的老乡们来扁您，您可能牺牲在北京街头。 您决定在北京二环路各黑市倒卖各种物品来发财致富，不仅还掉债务，而且要荣登北京富人排行榜. 您只能在北京呆40天，每次奔走一个黑市算一天。 您会在北京遭遇到各种事件，让您感到北京生存环境之恶劣、环境污染之剧烈、浮生之艰难，世态之炎凉。

“北京浮生记”界面简洁、可玩性高，趣味性强，具有北京浓厚的本土背景和辛辣的搞笑风格。 由于玩一次时间较短，它是您在工作学习之余消遣休闲理想的小游戏！ 游戏中的事件和物品价格都是随机的，给了您一个公平、公正和充满激情的一夜暴发机会。 您所需要的，是策略、胆量外加运气!

玩法

“北京浮生记”的玩法十分简单，很容易上手。基本操作过程如下：

  • 一、从一个地方（比如西直门）的黑市里买进适当的物品（可以是一种，也可以是多种）。
  • 二、奔走到另一个地方（比如复兴门）。你只需点击界面上的地点名就意味着你走到了那里。
  • 三、观察新地点的黑市。如果某物品的价格比您的买进价格高，就可以从您的出租屋里把这物品卖 出，这样您就赚了钱。再从新地点的黑市购入合适的其他物品。
  • 四、重复上述一、二、三的过程。

    • 最后，游戏清点您的总资产（现金+存款-债务）,计算您在北京富人榜里的排行。您的目标就是赚更多的钱，争取进入前十名。

要注意的是，您从一个地方奔走到另一个地方，在游戏里算一天，而您总共只能在北京呆40天。因此，要把握好 每一天，掌握好买卖策略，赚取最大利润。

所谓“入门容易、提高难。”游戏基本玩法很简单，但是要在40天里赚大钱（例如，突破3000万）也很不容易，很具有挑战性。需要您：

  • 一、对各种物品的价位有基本认识，这样才能“买低卖高”。例如，进口玩具价格在350元以下，买进就能赚钱。 如何把握物品的基本价位？这就需要您勤恳钻研，多玩几次，做一个有心人。
  • 二、充分把握游戏中的各种随机事件带来的商机。游戏里的很多随机事件会对物品的价格产生影响。例如， “市场上充斥着来自福建的走私香烟! ”事件会使香烟价格低迷，这时可以大肆购入进口香烟，说不定第二天 就能够赚大钱。又例如，发生“北京的富人疯狂地购买走私汽车！价格狂升!”事件时，应该立刻抛出走私汽车， 肯定能狂赚。
  • 三、及时扩大生意规模。这有两个含义：一是在您倒卖便宜物品（如盗版VCD）、资金有了一定的积累之后，要转向倒卖贵重物品，如 《上海小宝贝》、走私汽车等，它们的价格变化空间大，能够让您有机会赚大钱；二是当您金钱足够时，要 去租房中介那里租更大的房子，因为一开始您的房子只能够放100个物品，为了把生意做大，就应该租能够放 更多物品的房子。
  • 四、运用其他策略，详细情况请参考策略部分。

游戏中的事件

随机事件让“北京浮生记”充满了随机性、挑战性和趣味性。每次玩，情况都完全不一样。游戏中有三类随机事件：

  • 一、商业事件，它影响物品的价格和您带的物品数量。比如，“北京的大学生们开始找工作，水货手机大受欢迎！”事件 就能够让手机价格猛涨7倍。
  • 二、健康事件，它影响您的身体健康程度。比如，北京的沙尘暴就会让您健康水平下降。
  • 三、亏钱事件，能够让您辛苦挣来的金钱减少。例如，带红袖章的北京老太太会因为您是外地人而罚您的款，地铁口 扮演成乞丐的人又会使善良的您捐出一大笔钱。

在这种情况下，如何利用商业事件发财，避免亏钱事件就成为了游戏中很重要的策略。

界面简介

游戏界面很简单，主要由下面几个部分组成：

  • 一、黑市。里面有各种物品及其价格。
  • 二、您的出租屋。里面放着你购进的物品及其价格
  • 三、买进按钮。能够把您选中的黑市物品买进。
  • 四、卖出按钮。能够把您选择的出租屋里物品卖到黑市上。
  • 五、您的状态。包括您的金钱、存款、身体健康程度、您的名声。
  • 六、生意场所。是北京二环路的西直门、复兴门、和平门、崇文门、北京站、建国门、东直门和积水潭。您用鼠标点击任意地点，就代表您去了那里。
  • 七、重要场所。包括银行、医院、邮局、租房中介和网吧。它们的功能前面已经介绍。
  • 八、实时新闻。在屏幕最下面，是北京的实时新闻。
